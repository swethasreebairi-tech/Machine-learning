# 🧠 Mastering SVM Kernels: From Linear Limits to Non-Linear Power

This repository contains a complete machine learning tutorial demonstrating how Support Vector Machines (SVMs) can be used to solve non-linear classification problems using kernel methods and hyperparameter optimisation.

---

## 📌 Overview

Traditional linear classifiers struggle with non-linear data. This project explores how SVM kernels transform feature space to enable accurate classification.

We focus on:
- Comparing Linear, Polynomial, and RBF kernels
- Understanding decision boundaries visually
- Optimising model performance using GridSearchCV
- Evaluating results using accuracy and classification metrics

---

## 🎯 Objectives

- Demonstrate limitations of linear models
- Explore the impact of kernel choice
- Apply hyperparameter tuning (C and gamma)
- Visualise decision boundaries for better understanding
- Build an accessible and reproducible ML workflow
